Draft cube with your friends on discord!

Outline:
  1. Mod enters setup command with cubecobraid
  2. Players enter command to join draft
  3. Mod enters start command
  4. Bot DMs everyone their starting pack, 15 separate images
  5. Player reacts to which card they want, thumbs up adds to mainboard, ___ adds it to sideboard
  6. Once bot has received all picks, DMs each player the next pack
  7. Once draft is finished, bot DMs each player a text file with their draft pool separated into mainboard and sideboard
